Hound

A thoughtbot Product - Winter 2015

Hound is a developer tool that automates code style-checking, helping developers save time during code reviews while promoting a cleaner and more consistently written codebase.

When a user activates Hound on a GitHub repo, Hound will review and comment on style violations for every pull request that is opened.

Hound is a SASS product designed, built and maintained by thoughtbot. In addition to product design and implementation I contributed to product planning, customer support and efforts aimed at growing our active customer base.
